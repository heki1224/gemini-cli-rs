# Gemini Code Assist スタイルガイド

## 言語

すべてのレビューコメント・指摘・サマリーは**日本語**で記述してください。
技術用語（関数名・型名・コマンド等）はそのまま英語を使って構いません。

## リリース PR のスキップ

PR タイトルが `chore: release` で始まる場合（release-plz が自動生成するリリース PR）は、
コードの詳細レビューは不要です。「自動生成されたリリース PR のため詳細レビューを省略します。」と
一言添えてサマリーのみ提供してください。

## レビュー方針

- Rust のコードは `cargo clippy` の警告・`cargo fmt` のフォーマット規約を基準にしてください
- セキュリティ上の問題（API キーの露出・インジェクション・バッファ未制限等）は必ず指摘してください
- 軽微なスタイルの好みは指摘不要です（MEDIUM 以上の重要度のみコメントしてください）
- テストが不足している場合は指摘してください
